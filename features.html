CPU profiler is used to diagnose performance issues: "lag", low tick rate, high CPU usage, etc.

It is:

  • Lightweight ‚Äì can be ran in production with minimal impact.
  • Easy to use ‚Äì no configuration or setup necessary, just install the plugin/mod.
  • Quick to produce results ‚Äì running for just ~30 seconds is enough to produce useful insights into problematic areas for performance.
  • Customisable ‚Äì can be tuned to target specific threads, sample at a specific interval, record only "laggy" periods, etc.
  • Highly readable ‚Äì simple tree structure lends itself to easy analysis and interpretation. The viewer can also apply deobfuscation mappings.
