Geyser allows Minecraft: Bedrock Edition clients to join Minecraft: Java Edition servers.

This enables crossplay between the two editions of the game.

The ultimate goal of the project is to allow Minecraft: Bedrock Edition users to join Minecraft: Java Edition servers as seamlessly as possible.

‚öôÔ∏è How does it work?

Geyser acts as a translator between the Bedrock client and the Java server in Minecraft.

It takes data from the Bedrock client and translates it into a format that the Java server understands, and vice versa.

Geyser is compatible with any modern Minecraft version and can be installed either as a plugin or as a standalone program.

Grief Prevention:

  • When a player appears to be building something nice outside his claim, he's warned and shown his claim boundaries.
